# 📊 InternSpark Data Analytics Internship

## 📌 Project Description

This repository contains the completed tasks for the InternSpark Data Analytics Internship. The objective of this internship was to perform Exploratory Data Analysis (EDA), create meaningful visualizations, analyze real-world datasets, and generate actionable business insights using Python.

The project includes three analytics tasks:

* Sales Performance Analysis
* Customer Behavior Analysis
* Website Traffic Analysis

---

## 🚀 Features

### 📈 Sales Performance Analysis

* Dataset exploration and cleaning
* Statistical analysis (Mean, Median, Mode)
* Sales trend visualization
* Category-wise sales analysis
* Region-wise sales analysis
* Business recommendations

### 👥 Customer Behavior Analysis

* Customer demographic analysis
* Purchase behavior analysis
* Age distribution analysis
* Category preference analysis
* Customer insights generation

### 🌐 Website Traffic Analysis

* Traffic source analysis
* Page view analysis
* Session duration analysis
* Website engagement analysis
* Conversion performance analysis

## 📊 Key Insights

### Sales Performance Analysis

* **Technology** category generated the highest sales.
* **West** region contributed the highest revenue.
* High-value products significantly impacted total sales.

### Customer Behavior Analysis

* Average customer age is **44 years**.
* **Male** customers formed the larger customer segment.
* Customer preferences revealed clear purchasing patterns.

### Website Traffic Analysis

* **Organic** traffic was the largest source of visitors.
* Average page views per session were approximately **5**.
* Average session duration was approximately **3 minutes**.
